For bugs, feature requests, and API design discussion, use the Timestamp GitHub repository:

- [Issues](https://github.com/hawkeye64/timestamp/issues)
- [Discussions](https://github.com/hawkeye64/timestamp/discussions)

When opening an issue, include the Timestamp version, runtime environment, input values, expected result, and actual result.
