Use GitHub issues for bugs and feature requests:

[Open a Timestamp issue](https://github.com/hawkeye64/timestamp/issues)

For bugs, please include:

- The package version.
- The input value or Timestamp object.
- The helper you called.
- The expected result.
- The actual result.
- Whether the issue appears in browser, Node, SSR, or all runtimes.
